# Code Example: Make Your Welding or Assembly Robot a Station in Tekla PowerFab

Get code

[Tekla PowerFab API Welding/Assembly Robot Demo Program](https://developer.tekla.com/sites/default/files/files/download-codefile/file/Tekla%20PowerFab%20API%20WeldingAssemblyRobot%20Demo%20dotNET6_0.zip "Open archive in new window")9.22 MB

This code example and video demonstrates how a robotic welder or assembly robot can integrate with Tekla PowerFab to receive a list of items that are ready for assembly and to update PowerFab when those items are complete and ready for the next station. The Visual Studio C# example code can be downloaded from the link above, and [the video can be viewed here](https://youtu.be/XqEPfyrAWCM).

The Tekla PowerFab API and the Tekla Partners Program were created to make a fully integrated fab shop a real possibility. Join the T[ekla Partners Program here](https://developer.tekla.com/tekla-partners-program) and let's make a fully integrated fab shop a reality.
