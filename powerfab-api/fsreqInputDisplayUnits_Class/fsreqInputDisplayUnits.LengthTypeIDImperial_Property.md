# fsreqInputDisplayUnits.LengthTypeIDImperial Property

2 - Feet, Inches
3 - Inches, 16ths
4 - Decimal Feet
5 - Decimal Inches (2 Places)
6 - Decimal Inches
7 - Feet, Inches, 16ths (Free-Form)
